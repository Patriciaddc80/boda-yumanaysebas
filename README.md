# 💍 Invitación Digital - Boda Yumana

Sitio web interactivo para la invitación digital de boda de Yumana.

## 🚀 Características

- Diseño elegante y responsive
- Cuenta atrás del evento
- Información de ubicaciones (ceremonia y celebración)
- Servicio de transporte
- Código de vestuario
- Formulario RSVP
- Animaciones y gifs decorativos

## 📋 Requisitos

- Node.js 18+ 
- npm o yarn

## 🛠️ Instalación

```bash
npm install
```

## 💻 Desarrollo

```bash
npm run dev
```

El sitio estará disponible en `http://localhost:5173`

## 🏗️ Build

```bash
npm run build
```

## 👁️ Vista previa

```bash
npm run preview
```

## 📊 Stack

- **Frontend**: React 18 + TypeScript
- **Build**: Vite
- **Estilos**: CSS puro
- **Fuentes**: Montserrat, Playfair Display, Palisade

## 📝 Estructura

```
src/
├── components/      # Componentes React
├── data/           # Datos de la invitación
├── App.tsx         # Componente principal
├── App.css         # Estilos globales
└── main.tsx        # Punto de entrada

public/            # Archivos estáticos (imágenes, SVG)
```

## 🎨 Personalización

Edita el archivo `src/data/invitation.ts` para cambiar:
- Nombres de los novios
- Fecha del evento
- Ubicaciones de la ceremonia y celebración
- Información de transporte
- Código de vestuario

## 🚀 Despliegue en Vercel

1. Conecta el repositorio de GitHub en Vercel
2. Vercel detectará automáticamente que es un proyecto Vite
3. El sitio se desplegará automáticamente
